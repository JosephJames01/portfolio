Sprint Timing System
This project is a high-precision, Arduino-based sprint timing system (protoype). It uses two Arduino modules (start and finish), wireless synchronization via RFM69HCW radios, a laser break detector, and an LCD display to accurately measure and display sprint times. Clock synchronization is achieved using a PI controller, inspired by IEEE 1588 PTP.
Features:
-sub millisecond high-precision
-laser break Detection
-set and gun sounds imitating race conditions
-screen displaying infomation
-100m+ of range line-of-sight

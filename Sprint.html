Sprint Timing System
Features This project is a high-precision, Arduino-based sprint timing system (protoype).
It uses three modules (start, finish and laser)
Clock synchronization is achieved using a PI controller, inspired by IEEE 1588 PTP.
• sub 0.01 precision
• laser break Detection
• set and gun sounds imitating race conditions
• screen displaying infomation
• 100m+ of range line-of-sight
Components
• RFM69HCW
• Vishay TEPT 4400
• Adafruit PAM8302A Class D
• CN0293 1602 LCD Display
• Elegoo UNO R3
• Tactile Button
• Miscellaneous ie battery holder,
resistor, capcitor, on switch, wires
• Plastic Enclosures
